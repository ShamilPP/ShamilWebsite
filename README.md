# 🚀 Shamil PP - Modern Portfolio Website

A supernatural, modern Gen Z portfolio website with mysterious animations and progressive reveals.

## ✨ Features

### 🎨 Design Elements
- **Black Theme**: Fully dark mode with Gen Z aesthetics
- **Glassmorphism**: Beautiful glass effects throughout
- **Gradient Accents**: Cyberpunk-inspired color schemes
- **Custom Cursor**: Interactive cursor with trail effect

### 🌟 Sections

1. **Matrix Entry** - "You're at the right time, right place" intro
2. **Mystery Section** - Progressive reveal of developer identity
3. **Arsenal** - Tech stack showcase with hover effects
4. **Timeline** - Professional experience journey
5. **Projects** - Portfolio showcase with glass cards
6. **Stats** - Animated statistics counters
7. **Philosophy** - Developer mantra and mindset
8. **Connect** - Social media links with beautiful cards
9. **The Reveal** - Final name reveal with glitch effects

### 🎭 Animations
- Glitch text effects
- Matrix rain background
- Floating particles
- Typewriter effects
- Smooth parallax scrolling
- 3D tilt effects on cards
- Smooth fade-in animations
- Pulse and glow effects

### 🎮 Interactive Features
- Custom animated cursor
- Smooth scroll navigation
- Keyboard navigation (Arrow keys)
- Navigation dots sidebar
- Scroll indicators
- Hover effects on all interactive elements
- Easter egg (Konami Code: ↑↑↓↓←→←→BA)

- **HTML5** - Semantic markup

- **Google Fonts** - Space Grotesk & JetBrains Mono

## 🚀 Getting Started

### Option 1: Direct Open
Simply open `index.html` in your browser. No build process required!

### Option 2: Local Server (Recommended)
```bash
# Using Python
python -m http.server 8000

# Using Node.js (if you have it)
npx serve

# Using PHP
php -S localhost:8000
```

Then visit `http://localhost:8000` in your browser.

## 🎯 Customization

### Change Colors
Edit the CSS variables in `style.css`:
```css
:root {
    --primary-color: #00ff88;      /* Main green */
    --secondary-color: #0088ff;    /* Blue accent */
    --accent-color: #ff0088;       /* Pink accent */
}
```

### Update Content
Edit the HTML sections in `index.html`:
- Personal info in Section 9 (The Reveal)
- Social links in Section 8 (Connect)
- Projects in Section 5
- Experience in Section 4

### Modify Animations
Adjust timing in `script.js`:
- Loading screen duration: Line 3
- Animation speeds: Throughout the file
- Particle count: Line 168

## ⚡ Performance

- **No external dependencies** (except Google Fonts)
- **Optimized animations** using RequestAnimationFrame
- **Debounced scroll/resize** events
- **Reduced motion** support for accessibility
- **Mobile optimizations** - Disabled heavy animations on mobile
